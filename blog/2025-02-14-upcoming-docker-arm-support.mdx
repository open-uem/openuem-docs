OpenUEM can be deployed using Docker containers hosted at [Docker Hub](https://hub.docker.com/u/openuem) as explained in [Docs](/docs/Installation/Server/docker). Right now OpenUEM containers were only ready for linux/amd64 but soon linux/arm64 will be added.
